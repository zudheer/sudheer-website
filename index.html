Architecture

FastAPI backend (4 uvicorn workers) behind nginx, PostgreSQL + Redis, a cron container running APScheduler for subscription expiry. Firebase phone auth; dual payment verification through Stripe and Google/Apple IAP so subscriptions work regardless of store.

Librarian Assist

Side project · su-do.in
Live on Play Store

Library management for Kerala's libraries — books, members, lending, events, and finance.

Architecture

Offline-first Flutter app — local SQLite is the source of truth. Bulk Excel import matches the standard KSLC data-entry format with fuzzy header matching. Optional backup writes directly to a folder the app creates in the user's own Google Drive (scoped drive.file access, no server run by me).

FlutterSQLiteGoogle Drive API
Play Store ↗ librarian.su-do.in ↗
